SOLID - LISKOV SUBSTITUTION PRINCIPLE (LSP)
---------------------------------------------------------------------
STATEMENT:

DERIVED CLASSES MUST USABLE THROUGH THE BASE CLASS INTERFACE
WITHOUT THE NEED FOR THE USER TO KNOW THE DIFFERENCE

---------------------------------------------------------------------
ENUNCIADO:

LAS CLASES DERIVADAS DEBEN SER UTILIZABLES A TRAVES DE LA INTERFAZ
DE LA CLASE BASE SIN NECESIDAD DE QUE EL USUARIO CONOZCA LA DIFERENCIA.
---------------------------------------------------------------------
